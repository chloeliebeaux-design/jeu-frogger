# Jeu Frogger en Java - Projet étudiant

Petit jeu développé en Java dans le cadre de mes études.

J'ai crée le jeu Frogger en Java (programmation orientée objet, gestion des collisions, interface graphique, événements clavier) dans la cadre de ma formation. 

Fonctionnalités:
- Déplacements du personnage
- Gestion des collisions
- Système de chrono
- Choix de niveaux possible
- Interface simple et intuitive
- Architecture basée sur plusieurs classes
